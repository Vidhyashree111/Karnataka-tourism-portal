# Karnataka Tourism Portal

A web-based tourism portal for exploring Karnataka 
destinations with structured information and easy navigation.

## Team Members
- Vidhya - Full Stack Developer
- Member 2 - Full Stack Developer

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Java, JSP
- Database: MySQL

## Features
- User Login and Registration
- Destination Gallery
- Booking System
- Easy Navigation

## How to Run
- Download the project
- Import database in MySQL
- Run on Apache Tomcat Server
- Open browser and go to localhost
