# Karnataka Tourism Portal

A web-based tourism portal for exploring Karnataka destinations with structured information and easy navigation.

## Team Members
- Vidhya – Full Stack Developer  
- Member 2 – Full Stack Developer  

## Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: Java, JSP  
- Database: MySQL  

## Features
- User Login and Registration  
- Destination Gallery  
- Booking System  
- Easy Navigation  

## How to Run
1. Download or clone the repository  
2. Import the database into MySQL  
3. Configure Apache Tomcat Server  
4. Deploy the project on Tomcat  
5. Open browser and run
